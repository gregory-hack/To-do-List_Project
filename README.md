📌 Sobre o Projeto

Projeto de Aprendizagem para a Disciplina de Fundamentos de Programação WEB. 
Criação de uma To-Do List para gerenciamento de tarefas diárias, utilizando as tecnologias de HTML, CSS E JavaScript Puro.
O objetivo deste projeto, além de aprimorar meus conhecimentos nas linguagens citadas, é também
permitir que o usuário consiga:

✅Adiconar Tarefas e Editá-las.

✏️Elencar sua prioridade.

🗑️ Excluí-las.

💻Pesquisar tarefas



💡Tecnologias Utlizadas:

   - HTML5
 
   - CSS3
 
   - JavaScript

   - Fonte Awesome - ícones


 ⚙️ Observações:

 - Dados Salvos no LocalStorage
 - Projeto feito para Estudo
